SITE PURPOSE - A comparison of pixel and voxel based videogames including information about each game and why the viewer might choose to play one over another.
SITE NAME - Squares Compared - I chose this name because it conveys the purpose of the site well, as decribed above.

SCENARIOS

- Which pixel/voxel based sandbox game should I choose to play?
- Where can I find more information about [game]?
- Where can I play [game]?

COLOR SCHEMA

-primary color for headings and titles will be green
-secondary color for highlights, footer, and accents will be brown
-backgrounds for each page in the site will be hero images that match the theme of the game
--the primary and secondary colors were chosen to represent earth and trees, as these are a sort of reoccurring theme in these sandbox survival games.

TYPOGRAPHY - I chose Ubuntu and Arial

Most of the content will be in the other pages that I will create in the full project. This home page will just have a description of the site's purpose and minimal content compared to the other pages.
